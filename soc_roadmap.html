00
NETWORKING — THE DEEP END
Not just "how packets work" — learn to read the internet like a book
WEEKS 1–3 FOUNDATION ⚡ CRITICAL
›
Why this matters for SOC: 90% of alerts you'll investigate are network-based. If you can't read a packet capture, reconstruct a TCP stream, or spot a DNS tunnel — you're blind. This isn't "networking basics." This is network forensics training.
WHAT TO MASTER
  • OSI model — not memorization, but WHY each layer exists
  • TCP/IP — 3-way handshake, flags (SYN, ACK, RST, FIN), sequence numbers
  • DNS — full query lifecycle, record types (A, AAAA, MX, TXT, CNAME), DNS over HTTPS
  • HTTP/HTTPS — methods, status codes, headers, cookies, TLS handshake
  • ARP — how it works and why it's abused (ARP spoofing)
  • DHCP — IP assignment lifecycle, DHCP starvation attacks
  • ICMP — ping, traceroute, how attackers use it for tunneling
  • Subnetting — CIDR notation, calculate ranges manually, understand /24 vs /16
  • Ports — top 100 ports by memory. Know what runs on 443, 445, 3389, 22, 80, 25, 53
  • Wireshark — filter syntax, follow TCP stream, reconstruct sessions, export objects
THM ROOMS (DO THESE)
  • Networking Fundamentals CURRENT
  • Intro to LAN
  • OSI Model
  • Packets & Frames
  • Extending the Network
  • DNS in Detail
  • HTTP in Detail
  • How Websites Work
  • Wireshark: The Basics
  • Wireshark: Packet Operations
  • Tcpdump: The Basics
  • What is Networking? (Pre-Security)
HANDS-ON MISSIONS
  • Install Wireshark on your machine. Capture your own browser traffic. Find your DNS requests. Find the TLS handshake. Find the 3-way TCP handshake.
  • Use tcpdump -i any port 80 while browsing HTTP sites. Try to read the raw HTTP request in the terminal.
  • Run nmap -sV localhost and understand every field in the output.
  • Do a DNS lookup with dig for any website. Trace the full resolution chain using dig +trace google.com
  • Use traceroute or tracepath — understand every hop. What is each node?
  • Set up a home lab: 2 VMs on your M1 (Kali + Ubuntu). Capture traffic between them.
